Release-652

Alpha 18: 嘭! *

*也因它是著名导演 迈克尔·贝 导演的版本而广为人知√

欢迎来到A18爆炸性的首秀! 我们会延续从A17开始的优化建筑的传统. 不过这次我们的侧重点是 拆 家而不是 造 了哦. 是哒 -- “拆除” 按钮又回来了… 还带着"嘭!"之类的音效. 也就是说你的建筑不再是消失, 而是 被摧毁. (这只是该功能的首次亮相 -- 更高端的动画和特效随后会加上哒.) 届时不管是在建的还是建好的, 你都可以把它们"嘭!"掉哦, 旧版本存档里的也可以哦!

也有隆重登场的 (终于!) 建筑被拆毁时不留坑了 (怎么办到的? 不造. 这很… 魔法) 但请注意哥布林营地留下的坑坑是不会填掉的 -- 但我们会尽快实现这个功能的.

等等! 还有别的呢! 该测还为新手玩家添加了初始教程, (虽然眼下只有崛起之都有这个东西). 随时欢迎意见及建议回馈. 很多反馈说需要一个教程, 然后我们就加了进去, 不知道教程是不是把所有基础操作都讲清了.

该测还新增了特色和功能, 预计可以给玩家更优质的游戏体验, 你现在可以看到自家炉民空闲时也会做各种各样的动作了, 甚至敌人也会哦! 试试看能不能趁他不备偷袭吧! 当然还有些功能可能你 并不能 看到 -- 但同样很重要的. 我们在让石炉收集大家各种玩游戏的数据 -- 小镇怎么发展啊, 职业怎么规划啊, 一个档玩多久啊, 等等. 这些数据可以让石炉变成受大家喜爱, 给大家惊喜的游戏. 你可以在加载界面选择参与与否, 当然很明显我们是希望你的参与的!

最后, 是一些对尤其mod制作者十分重要的A18更新, 如果你制作mod或者是一个mod使用者, 请仔细阅读下文中相关的更新记录.

本测详录:

建筑倒地, 瞬间爆炸.

  • 建筑拆除回来辣! 炉民现在 "拆家" 可是会很快的哦. 你可以回收到约70%的建材.
  • 将物品放置在 在建的蓝图或者已经建好的建筑里 就会把那件物品与蓝图结合, 如果物品被毁坏则炉民会重新在原地放置一个.
  • 你 (暂时) 无法往在建的建筑里放新东西.
  • 新的楼梯生成路径避免复杂建筑中出现火腿肠一样的一坨楼梯.
  • 修复一个导致脚手架建造在建筑内的漏洞.
  • 修复一些物品在支撑物不存在的情况下不掉落.
  • 部分大型建筑建造优化.
  • 现在脚手架会等待对应梯子被拆除后再将自己标记为"已拆除".
  • 修复重新加载游戏导致内存泄漏.
  • 修复物品不放置的漏洞.
  • 自建建筑拆毁自动填坑
  • 新增建筑拆毁特效
  • 修正建筑合体时纹路错位
  • 囊括家具摆放的建材消耗计算精确化
  • 追加高土窗及土黄格子窗(均为瑞娅)
  • 追加红土尖顶房(瑞娅)
  • 地板边缘可旋转
  • 移除平顶屋屋顶的 "最大高度" 选项，因为这一选项永远不支持平顶屋也不应该在这里
  • 如果真的要建造大型复杂的建筑 (像某几张图片里的一样), 请在编辑完成前使用该指令,也就是说在建筑计算之前使用, 不要让电脑进行任何不必要的运算.

美好生活乐悠悠.

  • 转职树上不可转职的职业改为黑白

  • 调整动作哈欠 - 上次调整后的打哈欠在加速时简直就是在庆祝
  • 商人有金币上限啦! 你只能卖给他们买得起的东西!

关于Mod制作者 (与旧版本mod使用者) 的重要更新: Alpha 18 contains certain updates designed to improve performance of the game overall. These updates include a bunch of mod-related improvements we have made to make your life and the game better. The minimum mod version number has been increased to 3 so outdated mods will no longer load. Please update your mods (and let us know if you have any trouble) -- games using mods that have not been updated will not see those mods loaded. These changes are effective immediately.
