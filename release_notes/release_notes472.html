Release 460


全新的工作台

工匠们现在有各职业特有的工作台了，而且不同物品的制作也需要不同的工作台。比如说除以前就有的工作台以外，纺织工人有织布机和纺锤；铁匠有熔铁炉和铁砧；厨师有石磨、炉子和大锅；木匠多了张木工桌而石匠则追加了一个石头底座。

像做其他东西一样，工作台也能在制作界面点击制造，而造出来以后也能像其他可摆放的物品一样放置在地上。同样的工作台可以同时摆放多个，并会在同职业中共享一个物品制作队列。也就是说，两个纺织工人可以同时使用一个(或两个)织布机来织布。

我们在制作界面上还新增了一个"顺序"标签，他可以显示出制作流程中的细节和一些使工匠无法继续制造任务的原因。至于如何点出制造界面，通过选择工匠也可以，选择工作台也可以，或者直接点击游戏下方任务栏中心门新添上去的"工匠"按钮。

新的世界生成方式

现在的炉石世界将展现出一个全新的面貌！绵延的山脉以及庞大的水体俨然已经成为这个世界的标志。山脉与山麓的边缘更是可以看到多层渲染的细节，松树也将在世界内生成。同时，一切与地形生成的参数都存在于.json文件中可供调节，以便新生态区系的制作。

翻译流程的简化

石炉已经开始走向全球了！现只需安装一个语言本地化的模组，便可通过游戏中的！现只需安装一个语言本地化的模组，便可通过游戏中的"设置→系统"菜单来试下语种的改变。并且所有的字符串现都存放在一个独立的文件夹里，这样可以大大减轻翻译的工作量。
