Alpha 19: 那天我们终于想起了瑞娅的后代!

一个瑞娅之裔和沙漠地形相关问题会得到长期关(宠)注(溺)的Alpha.

欢迎来到石炉A19~ 该Alpha旨在为极度-缺爱的瑞娅之裔及相应的沙漠区系带来福利.

A17中,我们首次引入了城镇"升级"的概念 -- 只要你可以满足皇家使者提出的挑战中的一个就可以. 该升级已经 -- 截至目前 -- 只有崛起之都有. 但是, A19来辣, 瑞娅之裔的孩子们也要赶上崛起之都, 用他们自己的特色.

当然 (照例!) 还会有其他激动人心的东西包含在本测中, 包括mod管理器的试水和炉民互动的测试.

以下是该测的新内容明细:

富饶的小镇, 请允许我献上对你们的祝福.

正值 奈瑞尔·伊斯坎德尔 在外考察知名度高的小镇. 伊斯坎德尔手上地图标注的下一个小镇已经出现在地平线的那端 -- 当然你要想被标在地图上也可以... 只要你的小镇足够富裕. 戴瑞尔询问你想不想把自己小镇变成地图上的一个交易站, 或是一个以虔诚的信仰而闻名遐迩的圣猴神龛, 又抑或是众人皆知,充满异域情调的制作商. 建一座雕像来体现你的诉求吧 -- 升级至荣耀的二级城镇!你会得到一系列的好处, 包括:
  • 进阶建筑模板
  • 仓库扩容
  • 新交易者
  • 新背景音乐

沙漠之曲

等等 -- 重新看看上面的上面那一行. 你们说二级城镇瑞娅也会有 “新背景音乐.” 也就是说… 对哒! 很对, 的确. 瑞娅有了自己专享音轨啦, 多谢Raj Mann (和我们自己的, 虽然很神秘的, Doug). 而且不仅仅是二级城镇才有 -- 你会听到一整套配得上沙漠王国这个称号的独特音乐. 慢慢享受吧!

每一天, 建筑系统都在从每个角度变得更好!

我们在每个Alpha中都不断在努力, 以提升玩家的 -- 当然还有你们的炉民的! -- 建筑体验. 下面是A19中的优化:
  • 建筑时间大大缩短, 炉民一锤子可以放好几个方块!
  • 脚手架和楼梯不要木头啦! 尽管少真实, 不过我们相信这样能够让建筑系统更平易近人.
  • 非玩家制造的地形缺口也可填补. 在与怪物的对抗结束后, 选择坑内的土堆点击填坑即可.
  • 建筑模板里的自定义保持置于最后.
  • Instabuild now destroys the scaffolding on the building.
  • 修复多方块建筑的一些问题.
  • 建筑擦除工具可在水下工作

还有, 还有.

A19还在两大领域大大改进了物品描述信息.

你会发现一些炉民在空闲的时候会生成对话气泡 -- 这是更多社交活动的第一步. 请注意 -- 现在! -- 泡泡里面还是空的. 但不代表他们竟无语凝噎; 相反, 是因为我们还在考虑他们对话讲些什么. 你认为呢?

同样第一次出现在游戏里的: 石炉Mod管理器! 开始界面设置按钮进去就看得到啦. 你可以看到模组标签就在设置的低部那里. 现在,你可以把有用的mod直接拖到mods文件夹里, 然后用管理器就可以随时启用或禁用你喜欢的mod (或不喜欢的) 炒鸡方便. 麻麻再也不用担心窝一天到晚把mod拖进来拖出去惹… 你的mod管理器会帮你完成这项工作. 同时, 如果某只mod版本号跟不上, 管理器也会停止加载 -- 这就有助于减少一类由于过期mod而产生的问题啦.

本特色还在企划中, 但很想马上把它加进去. 请让我们了解你的看法!

(重要注意事项: Mod管理器 仅 能从开始界面的设置中进去. 游戏进行中的菜单里的设置没有这个功能. 因为更改mod设置需要重启游戏; 也就是说游戏中无法更改mod.)

另外, 我们给AI表现进行了一个巨大的更新解决了一些懒惰或者脑死亡的炉民问题, 大城镇中的人物表现也得到了大大改善.

更多东东:

  • 暂时禁用民兵标签因为有漏洞存在并且制作团队认为民并不是一个好方案. 我们会考虑其他方案.
  • 在地层视图中若建筑可见, 建筑上的炉民, 物品不再消失. 以前他们如在房顶上的话, 就算建筑可见, 人物也会消失, 这样可能就会导致饿死之类的事情发生.
  • 修复地层和透视视图中挖掘和炉民路径的可视性问题.
  • The official Debugtools mod is now included with the game (but disabled by default). You can enable it using the settings menu in the title screen.
  • 当自动制作开启时, 陶艺师在建筑开始建造是会自动制作所需陶砖.
  • 草药师制作药剂经验获得量稍有提升.
  • 修复视角移动旋转时会抖动. 现右键旋转视角将优先于视角的移动.
  • 暂时禁用'普通商队' 的交易事件, 因为有漏洞的存在会导致预定了货物不回来拿. (即要求即时交易). 我们希望在不久的将来重新回顾整个交易体系以给玩家更好的交易体验.
  • 修复挖掘区域衔接错误问题.
  • 现铁匠锻造动画用的锤子将是职业道具而非工人用锤.
  • 更多物品描述.
  • 挖掘区域插入地形以免占用空地
  • 防止哥布林勒索超过十个同一件物品 可避免酋长要求69个治疗药剂的情况
  • 优化石魔像游荡的动作表现. 它们找目标很累的.
  • 修复炉民无法发现入侵防御区域的敌人.
  • 修复点着时移动油灯导致光影效果无法正确重置.
  • 修复加载是屏幕延伸至更大分辨率
  • 修复大锣的碰撞体积
  • 修复战队巡逻错误
  • 修复炉民被卡窗户
  • 修复擦除合并的道路设计图会给原地形造成实际的破坏
  • 其他
所以, A20之前, 好好感受瑞娅的魅力吧!
